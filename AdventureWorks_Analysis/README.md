# AdventureWorks Sales Data Analysis

## Project Overview

This project analyzes the AdventureWorks dataset to understand sales performance, product trends, and customer purchasing patterns. The goal is to extract meaningful insights from the data to support better business decision-making.

## Objective

* Analyze sales performance across products and regions
* Identify trends affecting revenue and product demand
* Explore customer purchasing patterns using data analysis techniques

## Dataset

The dataset used in this project is the AdventureWorks dataset, which contains information about products, sales transactions, customers, and regions.

## Analysis / Approach

* Performed data cleaning and preprocessing to ensure data quality
* Conducted Exploratory Data Analysis (EDA) to understand patterns and relationships
* Created visualizations to highlight trends and key performance indicators

## Key Insights

* Certain product categories contribute significantly to overall sales revenue
* Sales performance varies across different regions and time periods
* Some products show consistent demand, indicating strong market preference

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* SQL
* Power BI

## How to Run the Project

1. Download or clone the repository
2. Open the Jupyter Notebook file in Jupyter Notebook or Google Colab
3. Run the cells to reproduce the analysis and visualizations
